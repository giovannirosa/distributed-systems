Página dedicada à versão 1 do algoritmo de sistemas distribuídos VCube.

Código

Fontes: vcube1.c vcube1.h

O código foi escrito a partir do código do vring.c do Trabalho Prático 1. As seguintes alterações foram realizadas para cumprir os requisitos do Trabalho Prático 2:

  1. Adição da propriedade do cluster a ser testado por cada processo na rodada corrente:
    2. {% highlight c %} // Descritor do processo typedef struct { int id; // identificador de facility do SMPL int *state; // vetor de estados de cada processo bool tested; // se o processo testou na rodada atual int cluster; // cluster a ser testado } ProcessType; {% endhighlight %}
  2. Inclusão de uma biblioteca da função Cis (cisj.c,cisj.h) fornecida pelo professor para calcular o cluster s do processo i.
  3. Funções auxiliares para a simulação que estão descritas no vcube1.h
  4. O caso de teste foi modificado para calcular o cluster a ser testado pelo processo, iterando nele até encontrar um processo correto ou testar todos daquele cluster.
    5. {% highlight c %} case TEST: if (status(process[token].id) != 0) break; // se processo falho, nao testa token2 = token; printf("\n==========================================\n"); printf("Iniciando testes do processo %d\n", token); print_state(N, token); nodes = cis(token, process[token].cluster); print_cluster(nodes->nodes, token, nodes->size); j = 0; do { token2 = nodes->nodes[j]; t = status(process[token2].id); t_result = t % 2 == 0 ? "correto" : "falho"; printf("Processo %d testou processo %d no tempo %4.1f: %s\n", token, token2, time(), t_result); if ((t == 0 && process[token].state[token2] % 2 != 0) || (t == 1 && process[token].state[token2] % 2 != 1)) { if (process[token].state[token2] == -1) { process[token].state[token2] = t; } else { ++process[token].state[token2]; } printf("State[%d] atualizado para %d\n", token2, process[token].state[token2]); count_event_test(N, token, token2); count_event_discovery(N, token, token2, process[token].state[token2]); } if (t % 2 == 0) { // se par esta sem falha, verifica novidades check_state(N, token, token2, nodes->nodes, process[token].cluster, j); } } while (t != 0 && ++j < nodes->size); set_free(nodes); schedule(TEST, 30.0, token); print_state(N, token); process[token].tested = true; count_cluster(token, logN); printf("==========================================\n"); break; {% endhighlight %}
  5. Uma função para contar o próximo cluster a ser testado pelo processo:
    6. {% highlight c %} if (process[token].cluster == logN) { process[token].cluster = 1; } else { ++process[token].cluster; } {% endhighlight %}
  6. A função para transferência de estados foi adaptada para o VCube (versão 1). Agora só são transferidas novidades do restante dos processos do cluster que não foram testados naquela rodada:
    7. {% highlight c %} for (int i = j + 1; i < POW_2(cluster - 1); ++i) { if (process[token2].state[nodes[i]] > process[token].state[nodes[i]]) { transfered = true; printf("Novidade encontrada, transferindo state[%d]...\n", nodes[i]); printf("State[%d] atualizado para %d\n", nodes[i], process[token2].state[nodes[i]]); process[token].state[nodes[i]] = process[token2].state[nodes[i]]; count_event_discovery(N, token, nodes[i], process[token2].state[nodes[i]]); } } {% endhighlight %}
  7. A informação do cluster a ser testado pelo processo é resetada após um evento de falha:
    8. {% highlight c %} // se falha, reseta o vetor de estados e cluster, pois é falha crash if (type == 1) { for (int i = (token + 1) % N;; i = (i + 1) % N) { if (i == token) break; process[token].state[i] = -1; } process[token].cluster = 1; } {% endhighlight %}
  8. Melhorias de comentários e logs.
